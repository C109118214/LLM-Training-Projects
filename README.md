# LLM Training Projects 🚀
本專案包含 LLM 訓練、微調、分散式運行與 CUDA 優化示例。

## 功能
- LLM 預訓練 (GPT-2, LLaMA-2)
- LoRA 微調
- FastAPI 部署 LLM
- Deepspeed/Ray 分散式訓練
- CUDA/PyTorch 加速
